## Note

Conradi Smith is collaborating with Del Negro, Saha, and Picardo to

(1) sequence the transcriptomes of parafacial interneurons and identify highly expressed transcripts to define neuronal subtypes,

(2) map the borders of neurons whose subtypes were defined in step 1,

(3) interrogate their expiratory function via cell population-specific photostimulation and photoinhibition experiments in awake intact adult mice.

Specific Aim (1) is elaborated as follows:

We will test the hypothesis that pFL expiratory rhythmogenic neurons have a distinct molecular fingerprint that uniquely differentiates them from other parafacial cells, including chemosensitive neurons and glia, as well as the C1 cells. The presumptive pFL can be loosely defined by peptide receptor expression, glutamatergic transmitter phenotype, and the lack of Phox2B expression. We will utilize the droplet-based 10x Chromium Single Cell Gene Expression with feature barcoding technology to create a single-cell cDNA library from cell suspensions obtained from tissue sections of the pFRG. We will sequence their transcriptomes and apply genomic data science to define expression profiles for pFRG cell types and identify clusters with similar profiles

However, it remains to be determined exactly what type of data analysis we should use in order to identify distinct cell types, but I imagine it will involve differenetial expression analysis and clustering.
