# Markov Chains

This folder contains examples related to discrete-state continuous-time Markov processes and stationary distributions.

## Main Files

- `markov.m`
- `poeq_eigs.m`
- `poeq_fullrank.m`
- `poeq_leastsquares.m`

## What It Demonstrates

- simulating a Markov process from a generator matrix
- computing equilibrium or stationary behavior in multiple ways
- comparing linear-algebra approaches

## Best For

Students with some comfort in linear algebra and probability who want examples connected to stochastic processes.
