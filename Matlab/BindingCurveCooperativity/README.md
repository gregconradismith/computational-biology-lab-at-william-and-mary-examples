# Binding Curve And Cooperativity Example

This example explores binding curves and related transformed plots such as `y/(1-y)` and Hill-style slope interpretations.

## Main File

- `example_binding_curve_analysis.m`

## What It Demonstrates

- generating a logarithmically spaced input grid
- comparing different binding functions
- plotting transformed quantities to study cooperativity
- estimating a slope related to Hill-type behavior

## Best For

Students who are beginning to connect mathematical models to biochemical interpretation.
