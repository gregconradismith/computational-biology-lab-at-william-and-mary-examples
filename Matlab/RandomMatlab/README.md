# Random MATLAB

This folder has been lightly reorganized into topic-based subfolders so it is easier to browse.

## What This Folder Is

- a collection of ideas, experiments, utilities, and one-off examples
- a useful browsing location for advanced students looking for inspiration
- not a structured tutorial sequence

## Subfolders

- `DynamicalSystems`: simulation-oriented scripts and model behavior examples
- `ProbabilityStatistics`: probability, sampling, goodness-of-fit, and stochastic examples
- `GraphsCombinatorics`: graph, combinatorics, permutation, and discrete-structure examples
- `SymbolicAnalytical`: symbolic, algebraic, and more theoretical exploratory scripts
- `LearningDecision`: simple learning and decision examples
- `VisualizationGeometry`: plotting, geometry, and visual exploration examples
- `DelayDifferentialEquationForSTDPSimulations`: delay-differential and integral-equation materials
- `LocalGlobalFHN`: FitzHugh-Nagumo-related materials
- `PositionDeterminationMatlab`: position-determination modeling materials
- `SandpileMovies`: sandpile examples and animations
- `LeVeque-fdmbook`: reference material related to finite-difference methods

## Notes

- A few historical PDFs and generated figures still live at this folder level.
- Some older subfolders predated this reorganization and have been left in place.
- The goal of this pass is improved browsing, not a perfect archival cleanup.
