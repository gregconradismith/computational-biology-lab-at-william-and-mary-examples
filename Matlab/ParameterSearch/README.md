# Parameter Search Example

This folder contains a simple example of parameter exploration by repeatedly sampling a parameter, generating output, and saving figures to a timestamped directory.

## Main File

- `example_parameter_search.m`

## What It Demonstrates

- repeated trial generation
- random parameter sampling
- saving outputs to a dedicated results folder
- the basic logic behind a parameter sweep

## Best For

Students who are starting to think about computational experiments that involve many runs rather than a single simulation.
