# Numerical Integration Of A 2D Linear ODE System

This folder extends the scalar ODE example from `Matlab` to a two-dimensional linear system.

## Main File

- `exampleode2.m`

## What It Demonstrates

- solving a system of ordinary differential equations with `ode15s`
- plotting multiple state variables against time
- plotting trajectories in phase space

## Good Prerequisite

Work through `Matlab/NumerialIntegrationOfScalarODE/exampleode.m` first if you are new to ODE examples in MATLAB.
