# Game Of Life And Neighbors

This folder contains a cellular automaton example based on Conway's Game of Life.

## Main Files

- `mylife.m`
- `neighbors.m`

## What It Demonstrates

- grid-based state updates
- neighbor counting
- iteration over many generations
- sparse matrix ideas and visual display with `spy`

## Best For

Students who want a playful but meaningful example of discrete dynamical systems and simulation.
