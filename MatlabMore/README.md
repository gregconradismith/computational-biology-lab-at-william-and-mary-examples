# MATLAB More

This folder contains a broader collection of MATLAB examples, worksheets, and research-oriented scripts. It is best used after the smaller examples in [../Matlab/README.md](../Matlab/README.md).

## Suggested Entry Points

Good first folders in this section include:

- `ExampleMatlabWorksheets`
- `ExampleMatlabGameOfLifeAndNeighbors`
- `ExampleMatlabInterpolation`
- `ExampleMatlabBindingCurveCooperativity`
- `NumericalIntegrationOf2DLinearODESystem`
- `ExampleMatlabParameterSearch`

## Folders To Treat As Reference Or Archive

- `RandomMatlab`: miscellaneous scripts collected over time; useful for browsing, but not a structured student entry point
- `Example1DAdvectionReactionSchemeSave`: more specialized research code than beginner material
- `ExampleMatlabGPCRCycleKinetics`: likely most useful once you already know the biological context
- `ExampleMatlabImprovingFiguresAndPlots` and `ExampleMatlabImprovingFiguresAndPlotsDefaultSettings`: reference material for figure polish rather than first-pass learning
- `ExampleMatlabSpeedingUp` and `ExampleSpeedingUpMatlab`: overlapping performance-oriented reference folders
- `ExampleResearchNotesTikZMatlabIntegration`: better treated as a hybrid research-note example than a core MATLAB lesson

## Topics Represented Here

- numerical integration
- interpolation
- parameter estimation and parameter search
- Markov chains
- bootstrap confidence intervals
- symbolic plotting
- performance and figure improvement
- discrete and continuous modeling worksheets

## Notes For Students

- Some folders are polished teaching examples.
- Some folders are closer to research notes or utility scripts.
- Start with folders whose names clearly match what you want to learn.
- `RandomMatlab` is best treated as a miscellaneous archive, not as a structured starting point.
- `ExampleMatlabSpeedingUp` and `ExampleSpeedingUpMatlab` appear to overlap; use them as references, but they are not good first stops.
- If a folder does not yet have a `README.md`, inspect the main `.m` file and consider contributing a short description after you understand it.

## A Practical Learning Path

1. Work through the worksheet PDFs in `ExampleMatlabWorksheets`.
2. Try one modeling example such as `NumericalIntegrationOf2DLinearODESystem`.
3. Try one data-analysis or fitting example such as `ExampleMatlabParameterSearch`.
4. Use the more specialized folders only when they connect to your project.
