# HPC Sciclone

This folder contains examples related to high-performance computing workflows on William & Mary's Sciclone cluster.

## Current Example

- `RunningMatlabOnScicloneByDanBorrus`

## Who Should Start Here

This section is best for students who already have a reason to run MATLAB jobs on the cluster, such as parameter sweeps or long-running computations.

It is not the best starting point for students who are still learning MATLAB basics.

## Suggested Order

1. Learn the relevant MATLAB workflow locally first.
2. Read `RunningMatlabOnScicloneByDanBorrus/README.md`.
3. Adapt the cluster scripts only after you understand the local MATLAB script they are launching.
