# LaTeX Poster Examples

This folder contains historical poster examples from lab work and conference materials.

## Best Use

Treat this folder as a reference library, not as a single clean template collection.

Some folders here are polished examples of real posters. Others are older materials preserved because they may still contain useful layout ideas, figure arrangements, or package configurations.

## Suggested Use For Students

1. Start with the smaller LaTeX examples in [../README.md](../README.md).
2. Come here only after you are comfortable compiling ordinary LaTeX documents.
3. Copy the poster example closest to your needs into a separate project folder before modifying it.

## Notes About The Subfolders

- `Jana Poster Final BPS 2010`, `RyanPosterFinalBPS2010`, and `YanPosterFinalBPS2010` are best understood as historical real-world poster examples.
- `OldPosterThatNoLongerCompile` is explicitly archival and may require substantial repair before reuse.

## Maintainer Note

If this repository gets a future breaking reorganization, this folder would benefit from a split between:

- `Templates`
- `HistoricalExamples`
- `Archive`

For now, keeping the files in place and clarifying their purpose is the safer option.
