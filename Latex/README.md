# LaTeX

This folder contains LaTeX examples for mathematical writing, manuscripts, and posters.

## Good Places To Begin

- `ExampleLatexMinimal`: a small starting point
- `ExampleLatexSimple`: a straightforward manuscript example
- `ExampleLatex`: a richer manuscript example with bibliography and figure files

## Other Specialized Examples

- `ExampleLatexSimplePDFLatex`: a PDFLaTeX-oriented example
- `ExampleLatexPST-ODE` and `ExampleLatexPST-ODE2`: examples involving PSTricks and ODE-related graphics
- `ExampleLatexPosters`: poster-related materials

## Historical And Specialized Material

- `ExampleLatex Latexian` appears to overlap substantially with `ExampleLatex`.
- `ExampleLatexPosters/OldPosterThatNoLongerCompile` is useful as an archive, but not as a student starting point.
- Some poster folders are best treated as real historical lab examples rather than minimal templates.

## Conservative Cleanup Recommendation

For now, this folder is easier to improve with documentation than with renaming. Some subfolders have historical names, spaces, or overlapping purposes, but moving them would risk breaking habits and references unless done as a deliberate later project.

## Recommendation For New Students

If you are new to LaTeX, begin with the smallest example that compiles successfully on your machine, then move toward the richer manuscript templates as needed.

When adapting an example:

- keep the original example intact
- copy it into a new working folder for your own paper or report
- document any package or compiler requirements that turned out to matter
