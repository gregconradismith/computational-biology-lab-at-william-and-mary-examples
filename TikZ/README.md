# TikZ

This folder contains figure examples built with TikZ.

## What You Will Find Here

- graph examples
- ROC-analysis examples
- reaction-scheme examples
- recombination-figure examples
- mixed examples showing custom styles and data-driven graphics

## Where To Start

Good first folders:

- `ExampleTikZ`
- `ExampleTikZGraph`
- `ExampleFigTikZGraph`

These are useful when you want publication-quality diagrams that are editable in plain text and integrate well with LaTeX.

## Advice For Students

- Start from the example closest to the figure you want to make.
- Change one thing at a time and recompile often.
- Save your own project in a separate folder rather than rewriting a shared example in place.
