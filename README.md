# Computational Biology Lab @ William & Mary

This repository contains example code, notes, figures, and writing resources for research students in the [Computational Biology Laboratory](https://wmcbl.wordpress.com) at [William & Mary](https://www.wm.edu).

The repository is designed to be a teaching library rather than a single software package. Most folders contain small, self-contained examples that students can study, run, adapt, and extend for their own research projects.

## Start Here

If you are new to the lab or new to computational work, use this order:

1. Read [GIT-HELP.md](GIT-HELP.md) for the basic Git workflow used in this repository.
2. Read [STUDENT-PATHS.md](STUDENT-PATHS.md) to choose a learning path.
3. Read the folder-level `README.md` in the topic you care about.
4. Start with a small example before opening the larger collections.
5. Copy an example into your own working folder before making substantial changes.

## Recommended First Stops

### If you want to learn MATLAB

Start in [Matlab](Matlab/README.md). It now contains both the introductory examples and the larger collection of advanced or research-oriented materials.

Suggested order:

1. `NumerialIntegrationOfScalarODE`
2. `VariableNumberOfInputArguments`
3. `PlottingSubplotsAndMultipleYAxes`

After that, continue deeper into the larger example folders within `Matlab`.

### If you want to write papers, reports, or posters

Start in:

- [Latex](Latex/README.md) for manuscripts and equations
- [TikZ](TikZ/README.md) for figures and diagrams
- [Sagemath](Sagemath/README.md) for Sage/LaTeX integration

### If you want notebook-style or mixed-media notes

Start in [Notebooks](Notebooks/README.md).

### If you need cluster computing

Start in [HPC-Sciclone](HPC-Sciclone/README.md). This is more specialized and is not the best first stop for most new students.

## Repository Map

- [Matlab](Matlab/README.md): introductory MATLAB examples plus larger worksheets, research-oriented scripts, and reference materials
- [Latex](Latex/README.md): LaTeX examples for manuscripts and mathematical writing
- [TikZ](TikZ/README.md): figure and diagram examples
- [Sagemath](Sagemath/README.md): SageMath and SageTeX examples
- [Notebooks](Notebooks/README.md): notebook-based notes and examples
- [HPC-Sciclone](HPC-Sciclone/README.md): cluster computing examples
- [bin](bin): small helper scripts

## How To Use This Repository Well

- Treat the examples as templates, not as polished software packages.
- Prefer making a copy of an example for your own work rather than editing shared examples directly.
- Add a short `README.md` when you contribute a new example.
- Keep filenames descriptive so future students can find your work easily.
- If your example requires special software, data, or setup, document that near the example.
- Some folders are best understood as archives or historical references; check the local `README.md` before assuming a folder is a recommended starting point.
